# Blinkit Grocery Insights Dashboard

## Project Objective
To analyse Blinkit grocery sales data and identify revenue trends, outlet performance, and product category contribution using Power BI.

## Tools Used
- Power BI
- DAX
- Power Query

## Dataset

The dataset was cleaned and standardised in Excel and Power Query before visualisation.

Key transformation:
- Standardised Item Fat Content categories using conditional logic.
- Removed inconsistencies in categorical values.

## Key Insights
- Tier 3 outlets generate the highest overall sales.
- Medium-sized outlets contribute the largest revenue share.
- Supermarket Type 1 outperforms other outlet types in total sales.
- Fruits & Vegetables and Snack Foods are top revenue-generating categories.

## Dashboard Features
- KPI Cards (Total Sales, Avg Sales, Item Count, Avg Rating)
- Sales Trend Analysis
- Product Category Revenue Breakdown
- Outlet Size & Location Analysis
- Interactive Filters
